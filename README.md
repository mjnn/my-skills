# my-skills

> 马捷个人 Skill 云端仓库（GitHub 分发节点）
> 对齐 [Qoder Skills Hub](https://github.com/mjnn/my-skills) 域分类与 CATALOG 组织方式

## 仓库定位

| 平台 | 安装路径 / 方式 |
|------|----------------|
| **Cursor** | 克隆 `skills/<域>/<skill-name>/` 到 `.cursor/skills/` 或用户 skills 目录 |
| **HanaAgent** | `install_skill from github: mjnn/my-skills/skills/<域>/<skill-name>` |
| **Qoder CN** | 下载 `assets/<skill>-vX.Y.Z.zip` 解压到 `%USERPROFILE%\.qoder-cn\skills\` |

## 目录结构（对齐 skill-hub）

```
my-skills/
├── README.md
├── CATALOG.md              # 人类可读注册表（域分类）
├── registry.json           # 机器可读注册表
├── AGENTS.md
├── CONTRIBUTING.md
├── assets/                 # 发布 zip（单 skill 离线包）
├── docs/

├── skills/
│   ├── platform-engineering/
│   ├── general-office/
│   ├── development-testing/
│   └── project-management/
└── templates/
```
